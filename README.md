# LangChain Tutorial

A practical guide to mastering the fundamentals of LangChain and building intelligent workflows using LCEL (LangChain Expression Language).

---

## 📂 Project Structure

### Chapter 1: The Basics
* **Environment Setup:** Configuring `.env` files and API keys.
* **Core Components:** Working with Prompts, LLMs, and Output Parsers.
* **Simple Chains:** Building your first functional pipeline using the pipe (`|`) operator.

### Chapter 2: Advanced Chains
* **Parallel Chains:** Executing multiple tasks simultaneously using `RunnableParallel`.
* **Conditional Logic:** Implementing routing and branching to create dynamic decision-making paths.
